# Feature Baseline

These feature briefs define the first staged product slices for the repository backlog.

- `foundation-and-kernel.md` covers the initial service kernel and local execution baseline.
- `scenario-simulation.md` covers scenario pack loading, policy control, and event generation.
- `platform-and-gitops.md` covers cluster bootstrap and runtime platform composition.
- `replacement-proof.md` covers the conformance and service-swap proof point.

The backlog in `sprintctl` is grouped around these slices rather than around tooling silos.
